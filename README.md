A full-featured bookstore system in JavaScript with cart management, payment processing, discount system, and inventory alerts. Includes integration tests with Jest.

## Features
- Search books by title or author
- Calculate total price with tax
- Process payments (simulated)
- Update inventory after purchase
- Handle errors for:
  - Out-of-stock items
  - Payment failures
  - Book not found

## Advanced Features
- Discount System: Apply coupon codes like SAVE10, SAVE20
- Inventory Alerts: Notify when stock is low after purchase

## Installation
- Clone the repository:
  - git clone https://github.com/KunnikarB/bookstore.git

- Install dependencies: npm install
  
## Testing
- Run all integration tests using Jest: npm test
